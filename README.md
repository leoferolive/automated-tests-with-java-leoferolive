# Projeto de Testes Unitários

Este é um projeto prático para aprender e praticar testes unitários em um ambiente Java.

## Objetivo

O objetivo deste projeto é demonstrar como escrever testes unitários eficazes para as diferentes partes de um sistema Java, garantindo assim a qualidade do código e a robustez das funcionalidades implementadas.

## Estrutura do Projeto

- `src/`: Contém o código-fonte do projeto.
- `test/`: Contém os testes unitários para as classes do projeto.
- `README.md`: Este arquivo que você está lendo agora.

## Tecnologias Utilizadas

- Java
- JUnit (framework de testes unitários)
- Maven (controle de dependências)

## Como Executar os Testes

1. Certifique-se de ter o Java, JUnit e Maven configurados em seu ambiente.
2. Navegue até o diretório raiz do projeto.
3. Execute o seguinte comando para rodar todos os testes:
    ```
    mvn test
    ```

## Testes Unitários

O projeto inclui uma classe de teste `SimpleMathTest` localizada no diretório `test/`. Esta classe utiliza o framework JUnit para testar os métodos da classe `SimpleMath`, garantindo que eles produzam os resultados esperados para diferentes operações matemáticas.
